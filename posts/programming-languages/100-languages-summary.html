100 programming languages in one post

January 19, 2025

Now that I've finished writing code in 100 different programming languages, here is a summary of the (mostly lesser-known) languages I used, along with links to my code and notes, unscientifically grouped into categories. (Obviously, some languages could fit into more than one category, but I decided to list each language only once.)

The languages I enjoyed the most were the following (in alphabetical order):

  • 8080 assembly: the language of the first home computer kit, the Altair 8800
  • Common Lisp: a pragmatic Lisp, with numerous compatible implementations
  • Factor: a modern concatenative/stack-oriented language
  • Futhark: data-parallel language that's easy to write
  • J: open source array-oriented language
  • Nim: honestly, the language I'm most excited to try again
  • Piet: 2D, pixel-based, stack-oriented esoteric language
  • Prolog: the most popular language for logic programming
  • Rebol: a tiny but powerful scripting language that didn't quite catch on
  • Tcl: a scripting language that is beautiful in its simplicity
  • Turbo Pascal: the world's tiniest integrated development environment
  • Unison: pioneering content-addressable code management/sharing
